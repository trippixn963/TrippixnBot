## Features

### Stats API
- Real-time server and developer stats
- Guild info: members, online count, boost level
- Bot status monitoring (TahaBot, OthmanBot)
- Developer presence via Lanyard

### AI Ping Responder
- Intercepts blocked developer pings
- GPT-4o-mini generates in-character responses
- Context-aware with current activity
- Webhook notifications to developer

### Media Tools
- `/download` - Download from Instagram, Twitter/X, TikTok
- `/translate` - Translate text to any language
- Reply commands for quick actions
- Auto-compression for large videos

---

## Commands

| Command | Description |
|---------|-------------|
| `/download <url>` | Download social media video |
| `/translate <text> [to]` | Translate text |
| `/languages` | Show supported languages |

### Reply Commands

| Reply With | Description |
|------------|-------------|
| `download` | Download media from message |
| `translate` or `tr` | Translate to English |
| `tr ar` | Translate to specific language |

## Tech Stack

- **Runtime:** Python 3.12
- **Framework:** discord.py 2.7+
- **AI:** OpenAI GPT-4o-mini
- **Downloader:** yt-dlp + FFmpeg
